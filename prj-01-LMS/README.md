# LMS (Learning Management System)

## 사용 기술
- spring boot 
- mariaDB


## 주요 기능
1. 회원 
- 사용자 : 
	- 회원 가입 및 인증메일
	- 로그인/로그아웃 : 사용자
	- 비밀번호 찾기(초기화)
	- 회원정보 수정
- 관리자 : 
	- 회원 목록 조회
	- 회원 상세 목록 조회
	- 회원 상태 및 비밀번호 초기화
2. 강좌
- 사용자 :
	- 강좌 목록
	- 강좌 상세 정보
	- 강좌 신청
- 관리자 :
	- 강좌 카테고리 : 조회/수정/삭제/정렬
	- 강좌 : 생성/수정/삭제, 스마트 에디터 적용
	- 강좌 신청 내역
